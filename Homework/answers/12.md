# Daily Database Backup with Gmail Alert — Complete Walkthrough

**Assignment:** Python script that backs up a database daily and sends a Gmail alert if backup fails 3 consecutive days.
**Level:** Basic — not production grade.
**OS:** Linux (Ubuntu)
**Alert:** Gmail via SMTP App Password

## Step 5 — Gmail App Password Setup

Regular Gmail password won't work — you need an **App Password**.

1. Go to [myaccount.google.com/apppasswords](https://myaccount.google.com/apppasswords)
2. Make sure **2-Step Verification** is turned on
3. Select app: **Mail** → device: **Windows Computer**
4. Click **Generate**
5. Copy the 16-character password → e.g. `abcd efgh ijkl mnop`

## Step 6 — The Script `db_backup.py`

```python
import os
import subprocess
import smtplib
import schedule
import time
from datetime import datetime
from email.mime.text import MIMEText

# ── SETTINGS — fill these in ──────────────────────────────────
DB_HOST     = "localhost"
DB_USER     = "backup_user"
DB_PASSWORD = "Backup@1234"
DB_NAME     = "backup_test_db"

BACKUP_FOLDER      = "backups"
FAILURE_COUNT_FILE = "failure_count.txt"
ALERT_THRESHOLD    = 3            # Send alert after 3 consecutive failures
BACKUP_TIME        = "23:55"      # Daily at 11:55 PM

YOUR_EMAIL    = "your.name@gmail.com"      # your Gmail (sender)
YOUR_PASSWORD = "abcd efgh ijkl mnop"      # 16-char Gmail App Password
ALERT_TO      = "your.name@gmail.com"      # who receives the alert
# ─────────────────────────────────────────────────────────────


def read_failure_count():
    if not os.path.exists(FAILURE_COUNT_FILE):
        return 0
    with open(FAILURE_COUNT_FILE, "r") as f:
        try:
            return int(f.read().strip())
        except ValueError:
            return 0


def write_failure_count(count):
    with open(FAILURE_COUNT_FILE, "w") as f:
        f.write(str(count))


def do_backup():
    os.makedirs(BACKUP_FOLDER, exist_ok=True)
    today = datetime.now().strftime("%Y-%m-%d")
    backup_file = os.path.join(BACKUP_FOLDER, f"backup_{today}.sql")

    command = (
        f'mysqldump --host={DB_HOST} --user={DB_USER} '
        f'--password={DB_PASSWORD} {DB_NAME} > "{backup_file}"'
    )

    result = subprocess.run(command, shell=True, capture_output=True)

    if result.returncode == 0:
        print(f"[{datetime.now()}] Backup SUCCESS → {backup_file}")
        return True
    else:
        print(f"[{datetime.now()}] Backup FAILED")
        print(f"Error: {result.stderr.decode()}")
        return False


def send_outlook_alert(consecutive_failures):
    try:
        msg = MIMEText(
            f"Hi,\n\n"
            f"The daily database backup for '{DB_NAME}' has failed "
            f"for {consecutive_failures} consecutive days.\n\n"
            f"Date: {datetime.now().strftime('%Y-%m-%d %H:%M')}\n"
            f"Database: {DB_NAME}\n"
            f"Host: {DB_HOST}\n\n"
            f"Please check the backup and fix ASAP.\n\n"
            f"-- Automated Backup Script"
        )
        msg["Subject"] = f"ALERT: DB Backup Failed {consecutive_failures} Days in a Row"
        msg["From"]    = YOUR_EMAIL
        msg["To"]      = ALERT_TO

        with smtplib.SMTP("smtp.gmail.com", 587) as server:
            server.ehlo()
            server.starttls()
            server.login(YOUR_EMAIL, YOUR_PASSWORD)
            server.sendmail(YOUR_EMAIL, ALERT_TO, msg.as_string())

        print(f"[{datetime.now()}] Alert sent to {ALERT_TO}")

    except smtplib.SMTPAuthenticationError:
        print("ERROR: Wrong email or password. Check YOUR_EMAIL and YOUR_PASSWORD.")
    except Exception as e:
        print(f"ERROR sending alert: {e}")


def run_daily_job():
    print(f"\n[{datetime.now()}] Running daily backup job...")

    success  = do_backup()
    failures = read_failure_count()

    if success:
        write_failure_count(0)
        print(f"[{datetime.now()}] Failure counter reset to 0.")
    else:
        failures += 1
        write_failure_count(failures)
        print(f"[{datetime.now()}] Consecutive failures: {failures}")

        if failures >= ALERT_THRESHOLD:
            print(f"[{datetime.now()}] Threshold hit — sending alert...")
            send_outlook_alert(failures)


if __name__ == "__main__":
    print(f"Scheduler started. Backup will run every day at {BACKUP_TIME}.")
    schedule.every().day.at(BACKUP_TIME).do(run_daily_job)

    while True:
        schedule.run_pending()
        time.sleep(60)
```

## Errors Faced and Fixes

| Error | Cause | Fix |
|---|---|---|
| `pip install` inside Python shell | Ran pip inside `>>>` prompt | Type `exit()` first, then run pip in terminal |
| `sudo apt install schedule` failed | `schedule` is a Python package not a system package | Use `pip3 install schedule --break-system-packages` |
| `pywin32` not working | Linux machine — pywin32 is Windows only | Switched to `smtplib` with Gmail SMTP |
| `Permission denied` on backups folder | Folder owned by root | `sudo chown $USER:$USER backups` |
| `SMTPAuthenticationError` | Regular Gmail password used | Generated Gmail App Password from myaccount.google.com |

---

## Key Concepts Used

| Concept | Where Used |
|---|---|
| `subprocess.run()` | Runs `mysqldump` shell command from Python |
| `failure_count.txt` | Persists failure count across days using a plain text file |
| `schedule` library | Triggers backup job daily at set time |
| `smtplib` + `starttls()` | Sends encrypted email via Gmail SMTP |
| Gmail App Password | Bypasses 2FA for script-based login |
| `chown` | Fixed Linux folder permission issue |
